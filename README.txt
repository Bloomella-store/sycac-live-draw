SYCAC Live Draw App - Final Bilingual v3

النسخة النهائية المطوّرة:
- واجهة عربية/تركية أكثر تنظيماً.
- سحب تلقائي لمدة 7 ثوانٍ.
- تباطؤ تدريجي قبل اختيار الفائز.
- تأثير رول/دولاب أسماء خلف خانة الفائز.
- اختيار الأسماء عشوائي من القائمة المتبقية، وليس حسب ترتيب ملف Kobo.
- لا يتم تكرار الفائز بعد التأكيد.
- تصدير الفائزين CSV وتقرير التدقيق JSON.

أفضل إعدادات تصدير Kobo:
1. Export type: CSV
2. Data value and header format: XML Values and Headers
3. Include fields from all versions: Yes
4. لا تعدّل ملف CSV في Excel قبل رفعه للنظام.

Kobo için önerilen ayarlar:
1. Export type: CSV
2. Data value and header format: XML Values and Headers
3. Include fields from all versions: Yes
4. CSV dosyasını Excel'de değiştirmeden yükleyin.
